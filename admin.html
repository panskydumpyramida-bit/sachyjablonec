Changelog

Historie změn a aktualizací webu

12. 4. 2026
v32

🔒 Bezpečnostní audit & Kvalita kódu

  • Helmet: Přidány bezpečnostní HTTP hlavičky — CSP, X-Frame-Options, HSTS, X-Content-Type-Options s whitelistem CDN zdrojů.
  • CORS: Odstraněna duplicitní konfigurace, origin: true nahrazeno ALLOWED_ORIGINS z env vars.
  • Sanitace vstupů: Nový sanitize-html na backendu pro komentáře, forum a zprávy. Na frontendu nová sanitizeHtml() utility pro zobrazení článků.
  • Hardcoded hesla: Odstraněno plaintext heslo ze seed.js — nyní z env var nebo náhodně generované.
  • Rate limiting: Globální API limiter 100 req/min.

🧪 Testy & Linting

  • Vitest: 33 unit testů pro auth middleware, RBAC hierarchii a sanitizační funkce.
  • ESLint + Prettier: Přidána konfigurace pro backend i frontend.
  • Race condition fix: Auth inicializace přepsána z 3× setTimeout na CustomEvent('auth:ready').

⚡ Performance & A11y

  • DB indexy: Comment (newsId, authorId), News (isPublished+date), ForumPost (topicId).
  • Batch standings: createMany místo cyklu create.
  • Keyboard dropdown: Enter/Space/ArrowDown/Escape + aria-haspopup, :focus-within.
  • WCAG AA kontrast: --text-muted #a0a0a0 → #b0b0b0 (5.57:1).
  • Smooth scroll: scroll-behavior: smooth + scroll-padding-top.
  • Optimalizace obrázků: Nový skript optimize-images.mjs (sharp) — PNG→WebP + komprese.

♟️ Komentování partií

  • Komentovat z editoru: Tlačítko 💬 u každé partie → otevře game-recorder → editace PGN s komentáři → uloží zpět do článku přes postMessage.
  • Sub-taby Partie: Admin sekce Partie má nově taby „Nahrané" a „Z článků".

🤖 AI Slovní anotace

  • AI komentář: Tlačítko ✨ v game-recorderu i game-vieweru. GPT-4o-mini generuje 1-2 věty česky srozumitelné dětem.
  • Endpoint: POST /api/ai/annotate — posílá FEN, tah, eval → vrací slovní hodnocení.

🎨 Admin UI

  • Stitch-inspired design: Timeline activity feed, ghost watermark na stat kartách, vertikální quick actions.
  • Login stránka: Vylepšený design s ikonami, placeholders, slide-up animací.
  • 404 stránka: Nová stránka s šachovým motivem.
  • jQuery: Sjednocen na 3.7.1 ve všech stránkách.
  • WebP: Management fotky na /about v <picture> elementech.

📖 Dokumentace

  • API Reference: Nový manuál docs/api-manual.html — kompletní dokumentace ~60 endpointů.
  • Admin manuál: Aktualizován na v32 — nové sekce Události, Diagramy, Komentování partií.

🐛 Opravy

  • CSP: Opraveny chybějící zdroje — unpkg.com (chessboard CSS), chess-api.com (engine), chess.com (embeds), mapy.com (mapa).
  • Fragmenty/diagramy: sanitizeHtml stripoval data-* atributy — odstraněna frontend sanitace na article contentu.
  • Puzzle Racer: Týdenní leaderboard počítá pondělí–neděle (ISO week).
  • Diagram reset: Opravena duplicitní reset() metoda.

🛡️ Atomické bloky – ochrana, drag & drop, edit modal

  • Atomické bloky: Diagramy a fragmenty se chovají jako chráněné objekty – nelze do nich psát ani je omylem smazat. Kliknutí otevře edit modal, Backspace/Delete maže celý blok najednou.

  • Rozvržení 3 sloupců: Nový layout 1/3 + 1/3 + 1/3 pro kompaktní rozmístění diagramů vedle sebe. Diagramy mají max-width 240px v třetinových sloupcích.
  • Reset tlačítko: Přesunuto z neviditelného board overlaye do viditelného tlačítka vedle popisku diagramu. Vždy viditelné, zvýrazní se při hoveru.
  • Varování: Při vložení fragmentu do 1/3 sloupce se zobrazí toast upozornění, že se fragment nevejde a je lepší použít širší layout.
